## Description

Kubernetes CLI To Manage Your Clusters In Style!. It uses vi/vim-like keybindings.

## Installation

1. Documentation and recent version of k9s is [here](https://k9scli.io/topics/install/).
2. Run the following commands
```sh
# Download installation script and run
curl -sS https://webinstall.dev/k9s | sh
# Check version
k9s version
```

## Uninstalling

1. Run uninstall script
```sh
curl -sS https://webinstall.dev/k9s/uninstall | sh
```
