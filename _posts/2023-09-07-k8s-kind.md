## Install

- Documentation and recent version of kind is [here](https://kind.sigs.k8s.io/docs/user/quick-start/#installation).
- Install docker. Check [post](/posts/docker) for installation instructions.

1. Install `kubectl`. Check [this file](kubectl_install.md) for installation instructions.
2. Download binary for `x86_64` or `arm64`. **version= 0.20.0**
   - `curl -Lo ./kind https://kind.sigs.k8s.io/dl/v0.20.0/kind-linux-amd64`
   - `#curl -Lo ./kind https://kind.sigs.k8s.io/dl/v0.20.0/kind-linux-arm64`
3. Make the binary executable
   - `chmod +x ./kind`
4. Move the binary to your `$PATH`
   - `sudo mv ./kind /usr/local/bin/kind`
5. Test installation
   - `kind --version`
